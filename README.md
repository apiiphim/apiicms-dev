# Hướng dẫn cài đặt
- Up lên host hoặc vps
- Giải nén
- Thay đổi thư mục upload-xóa đoạn này thành upload ( cái này vì e code trên máy k cho up cả ảnh :v)

- Chạy từ 7.4 tới 8.4
- Thay đổi quảng cáo: Thư mục gốc /static/player/artplayer/index.html
